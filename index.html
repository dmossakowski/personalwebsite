Line Text Scanner

Line Text Scanner


Android app for scanning text from images. It uses Google's MLKit. It solves a problem of many such apps which do not keep scanned text in lines. Google's MLKit finds text and provides it as an unstructured list which works well with continuous text paragraphs but behaves very badly with tabular data. This application solves this problem and makes scanning receipts or tables much easier.
